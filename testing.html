💡 Instrucciones

  1. Abre la consola del navegador (F12)
  2. Ve a tu panel de Webhook.site en otra pestaña
  3. Prueba aceptar/denegar el permiso de GPS
  4. Observa los requests que llegan al webhook
  5. Usa los botones arriba para probar el modal

🔒 Resetear Permisos de Ubicación

Para probar múltiples veces, necesitas resetear el permiso:

  • Chrome/Edge: Haz clic en el candado → Permisos → Ubicación → Restablecer
  • Firefox: Haz clic en (i) → Permisos → Acceder a tu ubicación → Limpiar
  • Safari: Preferencias → Sitios web → Ubicación → Eliminar
